# Set up printer at Atanasoff on mac

`printer` -> `add advanced`

* `type`: windows printer via spoolss
* `URL`: smb://papercut2.its.iastate.edu/Atanasoff-B01-LaserJet-9050
* `name`: whatever
* `location`: whatever
* `Use`: Generic PCL Printer

Then choose `duplifier`
